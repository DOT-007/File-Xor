About this bot

FILEXOR is a Telegram-based file streaming and download bot designed to avoid download delays by streaming files directly from Telegram without saving them to local disk. It provides a simple web interface for browsing and efficient chunked playback, plus utilities for server health, status, and quick admin actions.
